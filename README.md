# Sistema de Catálogo Digital com Checkout WhatsApp

Este é um sistema completo para venda de produtos personalizados.

## Funcionalidades
- **Catálogo Moderno**: Interface responsiva e premium.
- **Carrinho de Compras**: Adicione produtos e veja o total.
- **Integração WhatsApp**: Finalize pedidos com mensagens formatadas automaticamente.
- **Painel Admin**: 
  - Login seguro.
  - Gerenciamento de produtos (CRUD) com upload de imagens.
  - Personalização de cores, logo e número de contato.

## Como Executar

### 1. Iniciar o Backend
```bash
cd backend
npm install
npm run dev
```
*O servidor rodará em http://localhost:3001*

### 2. Iniciar o Frontend
```bash
cd frontend
npm install
npm run dev
```
*O site rodará em http://localhost:5173*
